# Desafio 2 — Integração com Banco de Dados e Segurança

## 📋 Descrição

Cansado de teoria? Pronto para colocar seus conhecimentos em prática? Então, aceite o
Desafio "Integração com Banco de Dados e Segurança" – uma oportunidade de aplicar tudo
o que você aprendeu e desenvolver soluções reais e inovadoras!

Você está trabalhando como desenvolvedor em uma agência de viagens que deseja criar uma
API para auxiliar os clientes no planejamento de suas viagens. A agência já possui um
site de reserva de viagens e uma estrutura de API para permitir que outras empresas de
turismo e aplicativos de terceiros integrem-se ao seu sistema.

A agência de viagens lida com informações sobre destinos, pacotes de viagens,
disponibilidade de hotéis e atividades turísticas. A API já foi produzida em um desafio
passado; no entanto, não possui integração com banco de dados e nem segurança aplicada.
Dessa forma, precisamos realizar a integração com banco de dados e mecanismos de
autenticação e autorização.

## 🎯 Missão

Trabalhar como desenvolvedor em uma agência de viagens que precisa de uma API poderosa
para auxiliar seus clientes no planejamento de viagens inesquecíveis.

### O desafio

- Integrar a API Spring Boot com o Banco de Dados PostgreSQL usando Spring Data JPA
- Implementar o Spring Security para garantir a segurança da API com:
    - Autenticação de usuários cadastrados no banco de dados
    - Autorização por perfil de acesso, controlando quem pode acessar quais recursos da API

---

## 📦 Resultado esperado

A API deve ser capaz de realizar todas as operações registrando em banco de dados
e validando o acesso por perfil de usuário cadastrado no banco de dados.

---

## ✅ Critérios de avaliação

- Integrou com Spring Data JPA
- Configurou corretamente a integração no arquivo de configurações
- Implementou corretamente as entidades
- Implementou corretamente os repositories
- Integrou corretamente o Spring Security
- Configurou corretamente as permissões de acesso
- Implementou a autenticação através de usuários cadastrados no banco de dados
